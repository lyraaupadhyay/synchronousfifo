A fully functional synchronous FIFO implementation with read, write, simultaneous read/write operations, and integrated count logic. Designed and verified for reliable data buffering in digital VLSI systems.

FEATURES:-

Synchronous operation using single clock domain

Full read/write functionality with independent pointers

Simultaneous read and write support

Count logic for occupancy monitoring (empty, full, and current count)

Parameterized design for flexible depth and width

Comprehensive testbench with directed and corner-case tests
